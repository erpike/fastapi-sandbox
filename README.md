### FastApi Sandbox project by Erpike

##### quickstart:

1. bash run:
   - `run.sh`
      - builds docker image
      - starts docker container
      - shows log

2. python run:
   - `guvicorn src.main:app`
     - flag: `--reload` (restart server on code update)

- available addresses:
  - http://127.0.0.1/docs
  - http://127.0.0.1/redoc
